# Dataset

## Download Instructions

This project uses the Forest Cover Type Prediction dataset from Kaggle.

**To run the code, you need to:**

1. Go to the Kaggle competition page:
   https://www.kaggle.com/c/forest-cover-type-prediction/data

2. Download the `train.csv` file (you may need to create a free Kaggle account)

3. Place `train.csv` in this `data/` directory

## Dataset Information

- **Source**: U.S. Forest Service
- **Observations**: 15,120 training samples  
- **Features**: 54 input variables (numerical and categorical)
- **Target**: Cover_Type (7 forest cover classifications)
- **Area**: Roosevelt National Forest, Northern Colorado
